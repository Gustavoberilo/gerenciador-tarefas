# Gerenciador de Tarefas

Sistema Full Stack de gerenciamento de tarefas (To-Do List), com autenticação de usuários e API REST.

## Tecnologias

* Node.js
* Express
* Prisma
* MongoDB
* React
* Vite

## Funcionalidades

* Cadastro e login de usuário
* Criação de tarefas
* Listagem de tarefas
* Exclusão de tarefas

## ⚙️ Como rodar o projeto

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
